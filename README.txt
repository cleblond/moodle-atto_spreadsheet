Moodle 2.7 Atto plugin for inserting DHTMLXSpreadsheet in moodle.
Requires moodle_filter_spreadsheet plugin.
Copyright 2014 onwards Carl LeBlond


Allows inserting of GPL dhtmlxspreadsheet (http://dhtmlx.com/docs/products/dhtmlxSpreadsheet/)
-Create editable or readonly spreadsheets
-Groups editing based on course group membership
